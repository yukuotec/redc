# REDC - Real Estate Data Collector

A focused real estate data collection tool for Shanghai Xuhui District.

## Features

- Single community focused approach
- Data collection for Shanghai Xuhui District
- CSV and JSON export capabilities
- Property information including price, size, location, and transportation access

## Usage

The tool generates real estate data for specific communities in Shanghai Xuhui District.

## Files

- `shanghai_xuhui_properties.json` - Property data in JSON format
- `shanghai_xuhui_properties.csv` - Property data in CSV format
